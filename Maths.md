
# Maths Cheatsheet

Power rules

- x^m * x^n = x^(m+n)
- x^m / x^n = x^(m-n)
- (x^m)^n = x^(m*n)
- x^0 = 1 (for any positive number x)
- x^-n = 1 / (x^n)
- x^(m/n) = the nth root of x^m
- (x*y)^n = x^n * y^n
- x = e^(log x) (for any positive number x)
- log(a*b) = log a + log b
- log(a^n) = n * log a
